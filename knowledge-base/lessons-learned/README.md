# Уроки, извлечённые из разработки

## 📋 Назначение

Эта директория содержит документы с уроками, извлечёнными в процессе разработки проекта GoldPC. Уроки помогают агентам избегать повторения ошибок и использовать проверенные решения.

## 📁 Структура

```
lessons-learned/
├── frontend/        # Уроки Frontend разработки
│   └── *.md
├── backend/         # Уроки Backend разработки
│   └── *.md
├── testing/         # Уроки тестирования
│   └── *.md
└── devops/          # Уроки DevOps и CI/CD
    └── *.md
```

## 📝 Шаблон документа урока

```markdown
# [Название урока]

**Дата:** YYYY-MM-DD  
**Спринт:** SX  
**Агент:** [Agent ID]  
**Модуль:** [Название модуля]

## Контекст

Описание ситуации, в которой возник урок.

## Проблема

Описание возникшей проблемы или вызова.

## Решение

Описание найденного решения.

## Урок

Краткая формулировка урока для запоминания.

## Код

Примеры кода (если применимо).

## Связанные документы

Ссылки на связанные контракты, паттерны или ADR.
```

## 🎯 Категории уроков

| Категория | Описание | Примеры |
|-----------|----------|---------|
| **frontend** | UI/UX, React, TypeScript | Оптимизация рендеринга, управление состоянием |
| **backend** | C#, ASP.NET, EF Core | N+1 запросы, транзакции, кэширование |
| **testing** | Unit, Integration, E2E | Mocking, тестовые данные, контрактные тесты |
| **devops** | Docker, K8s, CI/CD | Конфигурация окружения, деплой |

## 🔄 Автоматическое внедрение

Уроки автоматически внедряются в контекст агентов согласно конфигурации:

```yaml
agent_a:
  lessons_learned:
    path: "lessons-learned/frontend/"
    auto_inject: true
```
