## ❗️ ПРОБЛЕМЫ КОТОРЫЕ НУЖНО ИСПРАВИТЬ (ПРИОРИТЕТ #1)

### 🔴 Критические Блокеры
1. **Расчёт Питания и Производительности**
   - В данный момент расчёт либо не работает вообще, либо считает абсолютно неверные значения
   - Нет прогресс бара мощности
   - Нет проверки запаса БП
   - Нет расчёта bottleneck
   - Нет оценок FPS и производительности

2. **Модалка выбора компонента**
   - ❌ Ужасная компоновка фильтров: все сокеты в одну строку, нет переносов, нет отступов, невозможно использовать
   - ❌ Предпросмотр товара съехал за границы модалки, текст обрезается, нет вертикальных отступов
   - ❌ Фильтры показывают нулевые значения для всех сокетов кроме одного
   - ❌ Отсутствует сортировка по цене, популярности, рейтингу

3. **Слоты компонентов**
   - ❌ В пустых слотах отображаются лишние тексты сокетов `AM5 LGA1700` которые не должны там быть
   - ❌ Отсутствуют состояния ошибок и предупреждений
   - ❌ Нет inline бейджей совместимости
   - ❌ Отсутствуют иконки, неправильные отступы, не соответствует дизайну системы

4. **Правая панель сводки**
   - ❌ Не фиксируется при скролле
   - ❌ Нет расчётов, нет графиков, нет оценок
   - ❌ Цена обновляется с задержкой 2+ секунд
   - ❌ Отсутствуют кнопки сохранить, поделиться
