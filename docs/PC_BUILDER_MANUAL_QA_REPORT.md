## [БАГ] Фильтр брендов для материнских плат (и CPU) режет производителей по подстроке «amd» / «intel»

- **Сценарий:** 1 (п. 6), 4, 8  
- **Ожидание:** В модалке материнской платы в блоке «Бренды» видны производители платформы (ASUS, Gigabyte, MSI, ASRock и т.д.), фильтр по бренду не скрывает остальные чекбоксы без явного выбора пользователя.  
- **Реальность:** В [`PCBuilderPage.tsx`](src/frontend/src/pages/PCBuilderPage/PCBuilderPage.tsx) в `ComponentPickerModal` передаётся `restrictedManufacturerPlatform` для слотов `cpu` и `motherboard`, если по сокету определён `mfrPlatform` (`'amd'` для AM4/AM5). В [`FilterSidebar.tsx`](src/frontend/src/components/catalog/FilterSidebar.tsx) список брендов фильтруется так: для `'amd'` остаются только производители, у которых в **названии** есть подстрока `"amd"`. Производители вроде **ASUS**, **Gigabyte**, **MSI** из списка чекбоксов **исключаются** (API при этом отдаёт полный справочник — проверено запросом к `/api/v1/catalog/manufacturers`). Товары в сетке при этом могут отображаться с этими брендами — несоответствие чек-листу и здравому смыслу фильтра «производитель материнки».  
- **Скриншот:** не сохранялся; воспроизведение: выбрать CPU с AM4 → открыть МП → развернуть «Бренды».

## [БАГ] Ложное предупреждение об отсутствии встройки у CPU, если в API только `integrated_graphics` (snake_case)

- **Сценарий:** 6  
- **Ожидание:** Для Pentium G4400 с встройкой Intel HD Graphics 510 нет предупреждения «процессор не имеет встроенной графики» при отсутствии дискретной GPU.  
- **Реальность:** Детальный ответ API для товара `Процессор Intel Pentium G4400` содержит `specifications.integrated_graphics: "Intel HD Graphics 510"`, поле `integratedGraphics` отсутствует. В [`usePCBuilder.ts`](src/frontend/src/hooks/usePCBuilder.ts) в `checkBuildCompatibility` проверяется только `cpu.specifications?.integratedGraphics` (как boolean), без учёта `integrated_graphics` и строковых значений, в отличие от [`compatibilityUtils.ts`](src/frontend/src/utils/compatibilityUtils.ts) (`hasIntegratedGraphics`). В итоге возможно ложное предупреждение для CPU с заполненной только snake_case-спекой.  
- **Скриншот:** не приложен; проверка спеки:  
  `GET /api/v1/catalog/products/e8bf7a3a-d017-45a8-9349-bdb2efbfb020` → `integrated_graphics: Intel HD Graphics 510`.

## Наблюдения без регистрации как [БАГ]

### Сценарий 1 (основной поток)

- После выбора **AMD Ryzen 5 3600** и открытия **материнской платы:** в фильтрах отображаются сокет **AM4 (один вариант, отмечен)**, чипсеты **A520, B450, B550** (без Intel Z790/B660 и т.п.), тип памяти **DDR4** — **OK**.  
- Для **Ryzen 5 3600** в API указано `integrated_graphics: Нет` — предупреждение о отсутствии встройки **корректно**.

### Сценарий 3 (охлаждение / вентилятор)

- В коде заданы `typeFilter`: охлаждение — `['Башенный кулер', 'Жидкостное охлаждение']`, вентилятор — `'Корпусный вентилятор'`. Соответствие чек-листу зависит от строк в каталоге в поле `type`.

### Сценарий 5 (БП, ватты)

- В коде задаётся `wattage_min` в эффективных спеках и подсказка «Мин. XВт…» в [`FilterSidebar.tsx`](src/frontend/src/components/catalog/FilterSidebar.tsx). Полный UI-прогон не зафиксирован скриншотом.

### Сценарии 2, 7, 9

- **2 / 7:** Логика в [`compatibilityUtils.ts`](src/frontend/src/utils/compatibilityUtils.ts) и `ComponentPickerModal` — без полного пошагового прогона в браузере в этой сессии.  
- **9:** В модалке для выбранного товара отображаются превью с кнопкой «Увеличить фото»; искажение картинки отдельно не измерялось.

### Автоматизация браузера

- Некоторые клики по кнопке «Выбрать» на карточке перехватывались оверлеем; помогли клик с `offsetY=5` и поиск по названию. На реальном пользователе может не воспроизводиться.

## Рекомендации по исправлению (вне scope отчёта)

1. Не применять `restrictedManufacturerPlatform` к категории **motherboard** (или заменить на фильтр по фактическим брендам из фасетов категории, без подстроки AMD/Intel).  
2. В `checkBuildCompatibility` для iGPU использовать ту же логику, что и `hasIntegratedGraphics` в `compatibilityUtils`, включая `integrated_graphics`.
