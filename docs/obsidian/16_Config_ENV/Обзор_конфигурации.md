## 📄 Формат .env

```bash
# === Database ===
DB_USER=postgres
DB_PASSWORD=postgres
DB_HOST=postgres
DB_PORT=5432

# === JWT ===
JWT_KEY=your-production-jwt-key-min-32-characters!
JWT_ISSUER=goldpc
JWT_AUDIENCE=goldpc-users

# === SMTP ===
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USERNAME=noreply@goldpc.by
SMTP_PASSWORD=app-password-here
SMTP_FROM=noreply@goldpc.by
SMTP_FROM_NAME=GoldPC

# === Stripe ===
STRIPE_API_KEY=pk_live_xxx
STRIPE_WEBHOOK_SECRET=whsec_xxx
STRIPE_PUBLISHABLE_KEY=pk_live_xxx

# === Redis ===
REDIS_HOST=redis
REDIS_PORT=6379
REDIS_PASSWORD=

# === Monitoring ===
SENTRY_DSN=https://xxx@sentry.goldpc.by/1
APPLICATION_INSIGHTS_CONNECTION_STRING=InstrumentationKey=xxx

# === Docker Image ===
REGISTRY=ghcr.io
IMAGE_PREFIX=goldpc
IMAGE_VERSION=latest
```

## 🛡️ JWT Конфигурация

```jsonc
// Development
{
  "JwtSettings": {
    "Secret": "GoldPC_SuperSecretKey_ForDevelopment_Only_2024!", // ≥32 символа
    "Issuer": "goldpc",
    "Audience": "goldpc-users",
    "AccessTokenExpirationMinutes": 15,
    "RefreshTokenExpirationDays": 7
  }
}

// Production (OIDC)
{
  "JwtSettings": {
    "Authority": "https://auth.goldpc.by/realms/goldpc",
    "Audience": "goldpc-users",
    "RequireHttpsMetadata": true,
    "AccessTokenExpirationMinutes": 15,
    "RefreshTokenExpirationDays": 7
  }
}
```

## 🗄️ Connection Strings

| База данных | Dev | Prod |
|---|---|---|
| goldpc_catalog | `Server=postgres;Database=goldpc_catalog;Uid=postgres;Pwd=admin` | `Server=${DB_HOST};Database=goldpc;Uid=${DB_USER};Pwd=${DB_PASSWORD}` |
| goldpc_auth | `Server=postgres;Database=goldpc_auth;Uid=postgres;Pwd=admin` | `Server=${DB_HOST};Database=goldpc;Uid=${DB_USER};Pwd=${DB_PASSWORD}` |
| goldpc_orders | `Server=postgres;Database=goldpc_orders;Uid=postgres;Pwd=admin` | `Server=${DB_HOST};Database=goldpc;Uid=${DB_USER};Pwd=${DB_PASSWORD}` |
