### Database

| Переменная | Описание | По умолчанию | Обязательно | Сервис |
|---|---|---|---|---|
| `DB_USER` | Пользователь PostgreSQL | `postgres` | Да | Все |
| `DB_PASSWORD` | Пароль PostgreSQL | — | Да | Все |
| `DB_HOST` | Хост PostgreSQL | `postgres` | Да | Все |
| `DB_PORT` | Порт PostgreSQL | `5432` | Нет | Все |
| `ConnectionStrings__DefaultConnection` | Полная строка подключения | — | Да | Catalog, Orders, PCBuilder |
| `ConnectionStrings__PostgreSQL` | Строка для AuthService | — | Да | Auth |
| `ConnectionStrings__Redis` | Строка подключения Redis | `redis:6379` | Да | Catalog, Auth |
| `Redis__Host` | Хост Redis | `redis` | Да | Catalog, Auth |
| `Redis__Port` | Порт Redis | `6379` | Да | Catalog, Auth |
| `Redis__Password` | Пароль Redis | — | Нет | Все |

### JWT

| Переменная | Описание | По умолчанию | Обязательно | Сервис |
|---|---|---|---|---|
| `JwtSettings__Secret` | Симметричный ключ подписи | — | Да* | Auth |
| `JwtSettings__Issuer` | Эмитент JWT | `goldpc` | Да | Auth |
| `JwtSettings__Audience` | Аудитория JWT | `goldpc-users` | Да | Auth |
| `JwtSettings__Authority` | OIDC Authority URL | — | Да** | Auth |
| `JwtSettings__RequireHttpsMetadata` | Требовать HTTPS | `true` | Нет | Auth |
| `JwtSettings__AccessTokenExpirationMinutes` | TTL access токена | `15` | Нет | Auth |
| `JwtSettings__RefreshTokenExpirationDays` | TTL refresh токена | `7` | Нет | Auth |

> * — обязательно для development с симметричным ключом
> ** — обязательно для production с OIDC Keycloak
