## 🅰️ Сетевая и API терминология

| Термин | Расшифровка | Описание в контексте GoldPC |
|--------|-------------|----------------------------|
| **REST** | Representational State Transfer | Архитектурный стиль API. Все backend сервисы GoldPC предоставляют REST API на HTTP. |
| **gRPC** | gRPC Remote Procedure Call | Высокопроизводительный RPC-фреймворк (HTTP/2, protobuf). Используется для межсервисного вызова CatalogService ↔ OrdersService. |
| **BFF** | Backend for Frontend | Промежуточный слой API (GoldPC.Api), адаптирующий бэкенд-данные для фронтенда. Реализован через YARP reverse proxy. |
| **WebSocket** | — | Протокол полнодуплексной связи. Используется для SignalR хаба (уведомления в реальном времени). |
| **SignalR** | — | Библиотека ASP.NET Core для real-time веб-функций (WebSocket + fallback). |
| **JWT** | JSON Web Token | Токен доступа (`access_token`) и обновления (`refresh_token`). Используется для аутентификации пользователей. |
| **OIDC** | OpenID Connect | Протокол аутентификации поверх OAuth 2.0. Планируется для production (Keycloak). |
| **TOTP** | Time-based One-Time Password | Одноразовый пароль (RFC 6238). Используется для 2FA (Google Authenticator). |
| **BCrypt** | — | Алгоритм хеширования паролей. Используется в AuthService (`BCrypt.Net`). |

## 🅱️ Архитектурные паттерны

| Термин | Описание в контексте GoldPC |
|--------|----------------------------|
| **CQRS** | Command Query Responsibility Segregation. Разделение на чтение (`ReadOnlyCatalogDbContext`) и запись (`CatalogDbContext`). |
| **Outbox** | Шаблон для гарантированной доставки событий через БД. Реализован в OrdersService, но **отключён** ([[19_Tech_Debt/Архитектурные_проблемы]]). |

| **Factory Pattern** | `AuthDbContextFactory` — фабрика для создания контекста БД в миграциях. Содержит баг с портом ([[19_Tech_Debt/Архитектурные_проблемы]]). |
| **Repository** | Шаблон репозитория для доступа к данным. `IProductRepository`, `IOrderRepository`. |

## 🅲 Базы данных и инфраструктура

| Термин | Описание |
|--------|----------|
| **PostgreSQL 16** | Основная база данных. 7 баз данных (`goldpc_catalog`, `goldpc_auth`, `goldpc_orders`, `goldpc_pcbuilder`, `goldpc_services`, `goldpc_warranty`, `goldpc_reporting`). |
| **Redis 7** | Кэш + временное хранение (токены сброса пароля). |
| **RabbitMQ** | Брокер сообщений для MassTransit (OrderPlacedEvent, OrderPaidEvent). |
| **postgres_fdw** | Foreign Data Wrapper — расширение PostgreSQL для доступа к удалённым таблицам (используется в ReportingService). |
| **PITR** | Point-in-Time Recovery — восстановление БД на конкретный момент времени. |
| **WAL** | Write-Ahead Logging — журнал предзаписи для восстановления БД. |
| **Read Replica** | Реплика PostgreSQL только для чтения (используется CatalogService). |

## 🅳 Docker и оркестрация

| Термин | Описание |
|--------|----------|
| **Docker Compose** | Инструмент для запуска многоконтейнерных приложений. GoldPC использует 3 профиля: dev, prod, test. |
| **Blue-Green** | Стратегия zero-downtime деплоя. Blue = активный слот, Green = новая версия. Переключение через Nginx. |
| **Canary** | Постепенное переключение трафика (10% → 50% → 100%). **Не используется** в GoldPC. |
| **Zero-Downtime** | Деплой без остановки сервиса — достигается через Blue-Green. |
| **Kubernetes (k8s)** | Оркестрация контейнеров. **Планируется**, пока только Docker Compose. |
| **HPA** | Horizontal Pod Autoscaler — автоматическое масштабирование в k8s. **Не используется**. |

## 🅴 Мониторинг

| Термин | Описание |
|--------|----------|
| **Prometheus** | Система сбора метрик (Pull model). Собирает `/metrics` с каждого сервиса. |
| **Grafana** | Визуализация метрик (дашборды). |
| **Jaeger** | Распределённая трассировка (OpenTelemetry OTLP). |
| **Sentry** | Отслеживание ошибок в реальном времени. |
| **OpenTelemetry** | Стандарт для observability (метрики + трассировка + логи). |
| **OTLP** | OpenTelemetry Protocol — протокол передачи телеметрии (gRPC :4317). |
| **Serilog** | Библиотека логирования для .NET. |
| **Health Check** | Endpoint для проверки состояния сервиса (`/health`, `/health/live`, `/health/ready`). |

## 🅵 Аутентификация и безопасность

| Термин | Описание |
|--------|----------|
| **RBAC** | Role-Based Access Control — управление доступом на основе ролей (Client, Manager, Master, Admin, Accountant). |
| **2FA** | Two-Factor Authentication — двухфакторная аутентификация (TOTP). |
| **JWT** | JSON Web Token — токен доступа. |
| **Keycloak** | Identity Provider (OIDC) для production. **Планируется**. |
| **Rate Limiting** | Ограничение количества запросов. Настроено на Nginx (50r/s), **нет** в AuthService ([[19_Tech_Debt/Проблемы_безопасности#3-нет-rate-limiting-в-authservice]]). |
| **CORS** | Cross-Origin Resource Sharing. Настроен для `localhost:5173`, `localhost:3002`. |
| **CSP** | Content Security Policy — защита от XSS. Настроен частично. |

## 🅶 Платёжные и внешние системы

| Термин | Описание |
|--------|----------|
| **Stripe** | Платёжный провайдер. Интеграция через Stripe API + Webhook. |
| **Stripe Webhook** | Callback от Stripe при изменении статуса платежа. Эндпоинт: `POST /api/v1/webhooks/stripe`. |
| **SMTP** | Протокол отправки email. Используется для уведомлений (Gmail SMTP). |
| **MailKit** | .NET библиотека для работы с SMTP. |
| **X-Core** | Внешняя система учёта товаров. Используется для импорта данных (скрипт `seed-xcore`). |
| **SMS.ru / Twilio** | Провайдеры SMS-уведомлений. |

## 🅷 Проектно-специфичные термины

| Термин | Описание |
|--------|----------|
| **GoldPC** | Название проекта. Интернет-магазин + сервисный центр + конструктор ПК. |
| **PC Builder** | Интерактивный конструктор сборки ПК с проверкой совместимости. |
| **ServiceCenter** | Модуль сервисного центра (заявки на ремонт, диагностику, сборку). |
| **WarrantyCard** | Гарантийный талон (создаётся автоматически при заказе). |
| **GoldPC.Api (BFF)** | Backend for Frontend на YARP, порт :5008. |
| **CatalogDbContext** | Write-контекст для каталога (CQRS). |
| **ReadOnlyCatalogDbContext** | Read-контекст для каталога (наследует CatalogDbContext, что является "hack'ом"). |
| **AuthDbContextFactory** | Фабрика для миграций AuthService (содержит баг с портом 5432). |
| **OutboxMessage** | Сущность для Outbox паттерна (сообщения для отправки в RabbitMQ). |
| **MassTransit** | Библиотека для брокера сообщений (RabbitMQ). Отключена в CatalogService и OrdersService. |
| **PagedResult** | DTO для пагинированных ответов (содержит баг с TotalCount). |
| **LoggingHistory** | История входов пользователей (UserAgent всегда null — баг). |
| **XUnit** | Фреймворк тестирования C#. |
| **Playwright** | Фреймворк E2E тестирования (браузеры: Chromium, Firefox, WebKit). |
| **Testcontainers** | Интеграционное тестирование с Docker-контейнерами. |
| **k6** | Инструмент нагрузочного тестирования. |
| **Zustand** | State manager для React-приложения. |
| **TanStack Query** | Библиотека для управления серверным состоянием (React Query). |
