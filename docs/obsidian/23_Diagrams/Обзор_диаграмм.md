## 🔗 Использование Mermaid в Obsidian

```markdown
<!-- Все диаграммы используют mermaid.js -->
<!-- Блок-схемы (flowchart) -->
```mermaid
flowchart LR
    A --> B
```

<!-- Диаграммы последовательности (sequenceDiagram) -->
```mermaid
sequenceDiagram
    A->>B: Request
    B-->>A: Response
```

<!-- Круговые диаграммы (pie) -->
```mermaid
pie title Распределение
    "Frontend" : 40
    "Backend" : 40
    "Infra" : 20
```

<!-- Mindmap -->
```mermaid
mindmap
    root((Topic))
        Subtopic
```

<!-- Gantt -->
```mermaid
gantt
    title Timeline
    Task : 2026-05-01, 5d
```

<!-- Quadrant Chart -->
```mermaid
quadrantChart
    title Title
    x-axis "Low" --> "High"
    y-axis "Low" --> "High"
    "Item": [0.5, 0.5]
```
```
