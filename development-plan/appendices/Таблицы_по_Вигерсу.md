| Термин | Определение |
| --- | --- |
| Актор (Actor) | Пользователь или внешняя система, взаимодействующая с системой |
| Артикул | Уникальный идентификатор товара в каталоге |
| Гарантийный талон | Документ, подтверждающий гарантийное обслуживание товара или услуги |
| Конструктор ПК | Инструмент для подбора совместимых комплектующих с автоматической проверкой |
| Нефункциональное требование | Требование к качеству системы (производительность, безопасность и т.д.) |
| Пользовательская история (User Story) | Краткое описание функциональности с точки зрения пользователя в формате "Как [роль], я хочу [действие], чтобы [цель]" |
| Ролевая модель доступа | Модель безопасности, где права определяются ролью пользователя |
| Совместимость компонентов | Соответствие технических параметров комплектующих (сокет, тип памяти, мощность и т.д.) |
| Трассируемость требований | Возможность проследить связь между требованиями на разных уровнях и артефактами разработки |
| Use Case (Вариант использования) | Последовательность действий актора для достижения конкретной цели |
| Функциональное требование | Конкретная функция или поведение системы, которую необходимо реализовать |
| Экземпляр товара | Единица товара на складе (штука, комплект) |
