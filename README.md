**GoldPC** — веб-приложение для компьютерного магазина с сервисным центром. Платформа объединяет интернет-магазин комплектующих, онлайн-конструктор ПК и систему управления сервисным центром.

---

## 📋 Описание проекта

GoldPC — комплексная информационная система для автоматизации деятельности компьютерного магазина и сервисного центра. Проект разработан в рамках курсовой работы.

### Ключевые возможности:

- **🛒 E-commerce** — полнофункциональный интернет-магазин с каталогом товаров, корзиной и оформлением заказов
- **🔧 PC Builder** — интерактивный конструктор ПК с автоматической проверкой совместимости комплектующих
- **🛠️ Сервисный центр** — управление заявками на ремонт, диагностику и сборку ПК
- **📋 Гарантийный учёт** — автоматизация учёта гарантийных случаев

## ✨ Основные функции

### 📦 Каталог товаров
- Просмотр каталога без авторизации
- Фильтрация по категориям, производителю и цене
- Детальные технические характеристики
- Актуальные остатки и цены

### 🖥️ Конструктор ПК
- Выбор назначения сборки (игровой, офисный, рабочая станция)
- Автоматическая проверка совместимости компонентов:
  - Сокет процессора и материнской платы
  - Тип и частота оперативной памяти
  - Мощность блока питания
  - Форм-фактор корпуса
- Предупреждения о несовместимости
- Расчёт итоговой стоимости

### 📋 Управление заказами
- Оформление заказов с доставкой или самовывозом
- Онлайн-оплата и оплата при получении
- Отслеживание статуса заказа
- Email/SMS уведомления

### 🔧 Сервисный центр
- Заявки на ремонт, диагностику, сборку
- Назначение мастеров
- Отслеживание статуса работ
- Учёт использованных запчастей

### 🛡️ Гарантийный учёт
- Автоматическое создание гарантийных талонов
- Проверка срока действия гарантии
- Учёт гарантийных случаев

### 👥 Ролевая модель доступа
| Роль | Описание |
|------|----------|
| Гость | Просмотр каталога и конструктора |
| Клиент | Оформление заказов и заявок |
| Менеджер | Обработка заказов, управление товарами |
| Мастер | Выполнение ремонтных работ |
| Администратор | Полный доступ к системе |
| Бухгалтер | Финансовые отчёты |

## 🚀 Quick Start

### Предварительные требования

- [Docker](https://www.docker.com/get-started) 20.10+
- [Docker Compose](https://docs.docker.com/compose/install/) 2.0+

### Запуск проекта

```bash
# Клонирование репозитория
git clone <repository-url>
cd kursovaya

# Запуск всех сервисов
docker compose -f docker/docker-compose.yml up -d

# Проверка статуса
docker compose -f docker/docker-compose.yml ps
```

### Доступ к сервисам

| Сервис | URL |
|--------|-----|
| **Frontend** | http://localhost:3000 |
| **Catalog API** | http://localhost:5001/swagger |
| **PC Builder API** | http://localhost:5002/swagger |
| **Auth API** | http://localhost:5003/swagger |
| **Adminer (DB UI)** | http://localhost:8080 |

### Остановка проекта

```bash
docker-compose -f docker/docker-compose.yml down
```

## 🧩 Обновление каталога из X-Core (описание + юридическая информация)

Ниже — рабочий поток, чтобы обновить локальную БД данными со страницы X-Core, включая:
- структурируемое описание (берётся из `.detail_text`)
- юридическую информацию (берётся из `.description-prod__lists-item`: гарантия/адреса/импортёр/сервис)

### 1) Пересобрать `xcore-products.json`

```bash
cd scripts/scraper

# по умолчанию парсит базовый набор категорий
node index.mjs

# опционально: все категории
node index.mjs --all

# опционально: медленный режим
node index.mjs --slow
```

Результат будет в `scripts/scraper/data/xcore-products.json`.

### 2) Импортировать в БД (обновит и существующие товары)

```bash
# из корня репозитория
dotnet run --project src/CatalogService -- seed-xcore

# или полный сброс X-Core товаров + импорт + картинки (если готовы xcore-images.json)
dotnet run --project src/CatalogService -- seed-xcore-reset
```

Примечание: `CatalogService` при старте применяет миграции автоматически, поэтому новые поля для юридической информации будут созданы при запуске сервиса.
